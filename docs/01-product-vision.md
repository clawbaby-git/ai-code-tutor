# AI Code Tutor - 产品愿景

## 核心定位
基于 ArjanCodes 代码重构视频内容，打造 **Telegram 对话式 AI 编程导师**。

## 用户旅程（MVP 版）

```
1. 用户 Telegram 私信 AI 导师
2. AI: "欢迎！今天学习哪个课程？"
   -> 列出可用课程（如：logic）
3. 用户: "logic"
4. AI: "第1课/共8课。这是初始代码：[展示代码]
   你发现了什么问题？"
5. 用户用自然语言描述问题或要求修改
6. AI: 给出建议 -> 用户确认 -> 展示修改后的代码
7. 重复直到用户满意
8. AI: 对比最佳实践 + 学习总结
```

## 为什么选择 Telegram？

1. **低门槛**：用户无需安装新应用
2. **异步友好**：用户可随时中断、后续继续
3. **代码展示友好**：Markdown 代码块支持
4. **适合对话式学习**：自然语言交互最顺畅

## 三种修改模式的 MVP 选择

| 模式 | MVP 适用性 | 选择理由 |
|------|-----------|---------|
| 手动编辑 | 低 | 需要用户离开 Telegram |
| **自然语言指令** | **最佳** | 符合 Telegram 对话习惯 |
| 多选项选择 | 中等 | 可以作为辅助，但不作为主模式 |

**MVP 决策**：主打**自然语言指令模式**，AI 理解用户意图后生成修改建议，用户确认后应用。

## 课程配置的初步思路

当前：硬编码 logic 课程
未来：支持动态加载更多课程

```yaml
# courses/logic/meta.yaml
title: "代码重构：从混乱到清晰"
source: "ArjanCodes/examples/2025/logic"
lessons: 8
description: |
  学习如何一步步将嵌套地狱重构为清晰的规则引擎。
  涵盖：卫语句、条件命名、循环简化、逻辑数据化等技巧。
```

## 非目标（MVP 不做）

- [ ] 复杂的评分算法
- [ ] 多人在线协作
- [ ] 视频内容播放
- [ ] IDE 插件
- [ ] 自动批改代码

## 成功标准

1. 一个 Telegram 用户可以从头到尾完成 logic 课程的学习
2. 用户通过自然语言与 AI 互动，无需离开 Telegram
3. 每课结束后，用户能清晰看到自己的进步和学到的知识点
